# 📈 Stock-CIO

> AI 기반 개인 CIO(Chief Investment Officer) 투자 분석 및 의사결정 시스템

---

# Overview

Stock-CIO는 단순한 종목 추천 프로젝트가 아니다.

거시경제(Macro Economy)부터 개별 종목(Stock)까지 하나의 분석 체계를 구축하고, 이를 기반으로 투자 의사결정을 지원하는 AI 기반 개인 CIO 시스템이다.

프로젝트의 모든 분석은 동일한 프레임워크를 사용하며, 모든 투자 판단은 기록되고 검증된다.
