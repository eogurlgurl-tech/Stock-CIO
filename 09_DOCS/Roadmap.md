# Project Vision

STOCK-CIO는 개인 투자자를 위한 AI Chief Investment Officer(CIO)를 구축하는 프로젝트이다.

최종 목표는 데이터 수집부터 투자 의사결정, 포트폴리오 관리, 자동 보고서 생성까지 하나의 AI 플랫폼으로 통합하는 것이다.
