# STOCK-CIO

> **AI Chief Investment Officer**
> 개인 투자자를 위한 AI 기반 투자 분석 및 의사결정 지원 시스템

---

## Overview

STOCK-CIO는 단순한 종목 추천 프로그램이 아닙니다.

거시경제부터 시장, 섹터, 종목, 포트폴리오까지 하나의 분석 흐름으로 연결하여 투자 의사결정을 지원하는 AI CIO(Chief Investment Officer) 시스템을 구축하는 것을 목표로 합니다.
